# 🍕 Frontend da Pizzaria 🍕

## Iniciando

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Bem-vindo à documentação do frontend da Prosper Pizzaria! Este frontend interage com a API de Pizzaria para fornecer uma interface amigável para os usuários.
